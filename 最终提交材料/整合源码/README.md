# 立体多层停车场管理系统 — 演示说明

## 环境要求

- Windows 系统
- GCC 编译器（MinGW-w64）
- 终端编码 GBK（936）

## 文件清单

| 路径 | 用途 |
|------|------|
| `src/parking.h` | 头文件（全部结构体与函数声明） |
| `src/parking.c` | 源码（全部实现 + main 入口） |
| `parking.exe` | 编译后的可执行程序 |
| `data/` | 运行时数据（自动生成） |
| `tools/init_data.c` | 一键导入测试数据 |
| `tools/gen_test_data.c` | 仅生成停车测试数据（不含历史） |
| `tools/batch_exit.c` | 批量出场结算工具 |
| `README.md` | 本说明文档 |

---

## 一、编译

```bash
cd 整合源码

# 编译主程序
gcc -Wall -Wextra -finput-charset=UTF-8 -fexec-charset=GBK src/parking.c -o parking.exe

# 编译数据初始化工具
gcc -Wall -Wextra -finput-charset=UTF-8 -fexec-charset=GBK -DNO_MAIN -Isrc tools/init_data.c src/parking.c -o tools/init_data.exe
```

---

## 二、一键导入测试数据

```bash
./tools/init_data.exe
```

执行后自动完成三步：

```
第一步：注入 15 辆车（入场时间分布在 5分钟 ~ 24小时前）
第二步：全部出场结算 → 生成 15 笔交易记录，累计收入 269.00 元
第三步：注入 5 辆新车（刚入场，供后续测试用）
```

最终数据：

| 指标 | 值 |
|------|-----|
| 历史交易 | 15 笔 |
| 累计收入 | 269.00 元 |
| 当前在场 | 5 辆 |

---

## 三、启动系统

```bash
./parking.exe
```

## 四、功能演示路线

以 **管理员** 身份登录后，按以下顺序演示：

### 4.1 停车场概览

| 菜单 | 功能 | 预期结果 |
|------|------|---------|
| `6` | 车位状态总览 | 3层×4行×5列 ASCII 图，5个 ● 已占用 |
| `4` | 空闲车位查看 | 按楼层列出 55 个空闲车位 |
| `5` | 等待队列 | 当前无排队 |

### 4.2 收入统计（核心）

| 菜单 | 功能 | 预期结果 |
|------|------|---------|
| `7` → `1` | 总收入 | **269.00 元**，15 笔交易 |
| `7` → `2` | 今日收入 | 按当日入场时间统计 |
| `7` → `3` | 本月收入 | 本月累计 |
| `7` → `4` | 上周收入 | 上周一至周日 |

### 4.3 历史明细与分析

| 菜单 | 功能 | 预期结果 |
|------|------|---------|
| `8` | 历史交易明细 | 15 条记录，分页显示，含车牌/位置/时长/费用 |
| `9` | 楼层热度 | 3 层按周转率排名 |
| `10` | 峰值时段 | Top 3 入场高峰时段 |

### 4.4 出入场操作

| 菜单 | 操作 | 预期结果 |
|------|------|---------|
| `1` | 入场登记 | 输入新车牌 → 分配空闲车位 |
| `2` | 出场结算 | 输入在场车牌 → 显示费用并结算 |
| `3` | 查找车辆 | 查在场车位或等待队列位置 |

### 4.5 费用验证

选择 `2` 出场结算，依次测试以下车牌：

| 车牌 | 停留时长 | 预期费用 | 验证点 |
|------|---------|---------|--------|
| `沪D10001` | ~0分钟 | **0.00** | 刚入场，免费 |
| `京A00002` | — | 查历史 | 20分钟 → 9元 |
| `京A00010` | — | 查历史 | 24小时 → 40元封顶 |

> 注：京A 系列已全部出场（历史记录中），出场结算时用"沪D"系列测试。

### 4.6 用户管理

| 菜单 | 操作 |
|------|------|
| `13` → `1` | 添加操作员（如 `op1` / `op123`） |
| `13` → `1` | 添加车主（如 `user1` / `123`，绑定车牌 `沪D10001`） |
| `13` → `3` | 查看用户列表 |

退出后用新账号登录验证：
- **操作员** → 有入场/出场/查询权限，无收入统计
- **车主** → 仅可查看自己的车辆、费用、空闲车位、地图

## 五、计费规则

| 规则 | 值 |
|------|-----|
| 免费时长 | ≤ 15 分钟 |
| 起步价 | 5 元 |
| 小时费率 | 4 元/小时（不足1小时按1小时计） |
| 日封顶 | 40 元/天 |

公式：`费用 = min(5 + ceil((停留分钟 - 15) / 60) × 4, 40)`
